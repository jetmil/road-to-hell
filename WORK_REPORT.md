## Концепция дизайна

### Метафора
- **Спуск** — визуальная метафора падения/погружения
- **Распад** — элементы, которые "ломаются", глитчат, искажаются
- **Пустота** — минимализм, много чёрного пространства
- **Тлеющий огонь** — не яркий ад, а угасающие угли

### Цветовая палитра
```css
--void: #0a0a0a          /* Пустота - основной фон */
--ash: #1a1a1a           /* Пепел - вторичный фон */
--ember: #ff3d00         /* Тлеющий уголь - акцент */
--ember-glow: #ff6b35    /* Свечение угля */
--smoke: #666666         /* Дым - вторичный текст */
--bone: #e8e4e0          /* Кость - основной текст */
--rust: #8b4513          /* Ржавчина - декор */
```

### Типографика
- **Заголовки:** Bebas Neue (жёсткий, угловатый)
- **Текст:** Source Serif 4 (читаемый, классический)
- **Акценты/эпиграфы:** Courier Prime (машинописный, холодный)

### Уникальные эффекты
1. **Glitch-текст** — заголовки периодически "сбоят" (css/effects.css)
2. **Ember particles** — тлеющие частицы поднимаются снизу
3. **Text decay** — эпиграфы "трясутся" при наведении
4. **Scroll depth indicator** — индикатор глубины слева
5. **Burn transitions** — эффект прогорания при переходах
6. **Fracture lines** — трещины-разделители секций
7. **Noise/Scanlines** — лёгкий шум и линии развёртки
8. **Vignette** — затемнение по краям

## Уникальные особенности дизайна

1. **Индикатор глубины** — слева показывает % прочитанного, градиент от оранжевого к кровавому
2. **Тлеющие частицы** — медленно поднимаются снизу, как угольки
3. **Глитч-эффект** — заголовки периодически "сбоят" с красным/коричневым
4. **Шум и линии** — создают эффект старого монитора
5. **Виньетка** — затемнение по краям усиливает фокус
6. **Автоматическая подсветка** — ключевые слова (АД, катастрофа, тюрьма) подсвечиваются ember-text
